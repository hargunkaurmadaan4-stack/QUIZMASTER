# 🎯 Quiz Master

A Python terminal-based math quiz game.
## Features
- Addition
- Subtraction
- Multiplication
- Square
- Cube
- Random Questions
- Score Tracking
- Play Again
## Technologies Used
- Python
- Random Module
## Future Updates
- ⏱️ Timer Mode
- 🏆 High Score Saving
- 🎲 Mixed Question Mode
- 📊 Difficulty Levels
- 🎯 Custom Number of Questions
  ## 🚀 How to Run
1. Install Python.
2. Download the project.
3. Open the terminal.
4. Run: python quizmaster.py
